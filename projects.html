Micro-X Agent

github.com/StephenDenisEdwards/micro-x-agent-loop-python
Python / AI Agent

A general-purpose AI agent that turns natural language into action. Built on the Model Context Protocol (MCP), Micro-X is an interactive REPL that streams LLM responses in real time, executes tools in parallel, and keeps API costs under control with a multi-layer cost reduction architecture.

  • Multi-provider LLM support (Anthropic Claude & OpenAI GPT) with streaming responses
  • Parallel tool execution via asyncio, automatic retry with exponential backoff
  • 30+ MCP tool servers: Gmail, GitHub, LinkedIn, web search, WhatsApp, filesystem, and more
  • Layered cost reduction: prompt caching, on-demand tool discovery, conversation compaction, compiled mode
  • SQLite-backed session persistence with conversation resume, fork, and file checkpoint rewind
  • Continuous voice mode via Deepgram STT for hands-free operation
  • Trigger broker for scheduled autonomous agent runs via cron expressions

Interview Assist

github.com/StephenDenisEdwards/interview-assist-2
C# / Real-time AI

Real-time interview assistance application that captures audio, transcribes speech via Deepgram, and detects questions using LLM-based intent classification. Features a Terminal.Gui UI, session recording/playback, and an evaluation framework.

  • Real-time audio capture (microphone + system loopback) with live transcription via Deepgram
  • LLM-based intent detection pipeline for identifying questions and objections during live calls
  • Terminal.Gui console UI with headless mode and reporting capabilities
  • Session recording/playback system for offline analysis and testing
  • Ground truth annotation tooling and evaluation framework for measuring detection accuracy
  • Modular architecture with separate libraries for core logic, Windows audio, and UI
